# ShieldHer – Women’s Safety App (UI Design)

**ShieldHer** is a mobile UI/UX design created to ensure safe, reliable, and discreet transportation for women.  
The design focuses on **emergency alerts, real-time tracking, and trust-building features**, empowering users with control, privacy, and security - whether on public transport, in a tuk, or walking home.

# Problem Statement
Design a **user-friendly and discreet** UI/UX solution that ensures women’s safety during travel, focusing on:
- Emergency response and SOS triggers  
- Real-time journey tracking  
- Accessible and intuitive interface  
- Building trust and peace of mind
